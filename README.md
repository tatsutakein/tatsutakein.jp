## GetStarted

以下の手順を踏んで開発に参加しましょう。

### 1. Bootstrap

以下のコマンドを実行して開発環境を整えます。

```shell
$ make bs
```

### 2. `pnpm run dev`

`pnpm run dev`でローカルサーバーを立ち上げます。  
表示された URL にアクセスすると、サイトが表示されます。

```shell
$ pnpm run dev
```
